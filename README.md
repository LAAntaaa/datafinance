# Backtesting et analyse de performance de stratégies de trading simples sur actions US (Projet personnel)
## Description du projet:
Ce projet a pour objectif d'analyser la performance et la volatilité de stratégies de trading simples appliquées à des actions américaines.
L'analyse comprend le calcul d'indicateurs de performance (rendement annualisé, Sharpe ratio, drawdown, etc.) et le backtesting de statégies telles que:
- **Buy & Hold**
- **Croisement de moyennes mobiles (Moving Average Crossover)**
- **Stratégies basée sur le RSI (Relative Strength Index)**

---

## Objectifs
- Collecter et nettoyer les données financières avec 'yfinance'
- Calculer les principaux indicateurs de performance et de risque
- Backtester différentes stratégies de trading
- Comparer les performances et visualiser les résultats

---

## Technologies utilisées
- **Langage**: Python
- **Bibliothèques principales**:
  -'pandas', 'numpy'
  -'matplotlib', 'seaborn'
  -'yfinance'
  -'backtesting.py'
- **Environnement**: Jupyter Notebook
